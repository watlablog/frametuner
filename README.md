# FrameTuner

Browser-based video editor for trimming, cropping, resizing, tuning frame rates, removing audio, and exporting lightweight clips.

## Development

```sh
npm install
npm run dev
npm run typecheck
npm run build
npm run preview
```

## Current Scope

This repository is currently implementing Milestone 1 from `DESIGN_FrameTuner.md`:

- Vite + TypeScript static app
- WATLAB Tools-like dark shell UI
- Header, upload, preview, inspector, and export placeholders
- No video loading, FFmpeg, or editing logic yet
